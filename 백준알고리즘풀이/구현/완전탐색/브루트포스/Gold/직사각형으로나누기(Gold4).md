# 문제 유형
- 완전탐색
  - 주어진 직사각형을 3개로 나누었을 때 각각 작은 직사각형 합의 곱의 최댓값을 구하는 문제이므로, 3개의 직사각형을 만드는 경우의 수를 모두 탐색하여 최댓값을 구하면 됨

# 주요 코드 개념
- 직사각형의 1,1부터 현재 좌표 영역 내 모든 수의 합을 미리 계산해서 해당 좌표 칸에 저장

- 3개의 직사각형을 만드는 경우의 수(6가지)를 모두 탐색하면서 최댓값 계산
