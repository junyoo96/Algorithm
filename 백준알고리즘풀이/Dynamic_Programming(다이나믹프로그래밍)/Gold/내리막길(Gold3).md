# 문제 유형
- Dynamic Programming
  - DP + DFS 혼합 문제
    - 주어진 지도에서 출발지점에서 목표지점까지 항상 내리막길로만 이동하는 경로들 개수를 구하는 문제
    - DFS로만 풀면 이미 갔던 길에 대해서 중복연산 하기 때문에 시간초과가 발생하므로, DP를 같이 이용해서 풀어야함
