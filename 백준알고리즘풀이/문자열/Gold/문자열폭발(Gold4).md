# 시간복잡도
- O(N)
  - 범위가 다음과 같으므로, O(N)의 시간복잡도를 알고리즘으로 구현해야함
    - 입력 문자열의 범위 : 1~1,000,000
    - 폭발 문자열의 범위 : 1~36
  - 일반 문자열 함수 사용시, 시간초과  
  - stack 사용
    - 문자열을 추가, 제거하는 시간을 O(1)로 구현해야 하기 때문에 stack 사용
