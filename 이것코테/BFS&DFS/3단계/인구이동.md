# 문제 유형
- DFS or BFS
  - 모든 나라의 위치에서 상,하,좌,우로 국경선을 열 수 있는지를 확인해야 하기 때문

- 각 나라의 연합 번호를 저장할 2차원 리스트 생성
  - 방문 안한 나라는 -1로 표시
