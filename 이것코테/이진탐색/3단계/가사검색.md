# 문제 유형
- 이진 탐색
  - 각 쿼리 키워드에 매치되는 모든 단어의 개수를 세는 문제
    - 쿼리 키워드 개수(2~100,000) 이고 단어 개수(2~ 100,000)이므로 매치하는지 확인해야할 데이터의 개수가 매우 많기 때문에 완전탐색이 아니라 이진탐색을 사용해야함
