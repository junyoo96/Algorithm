# 문제 유형 
- 그리디
  - 먹는데 시간이 적게 걸리는 음식부터 확인하는 그리디 방식으로 해결할 수 있음
    - 모든 음식을 시간을 기준으로 정렬한 뒤에, 시간이 적게 걸리는 음식부터 제거해 나가는 방식 이용
