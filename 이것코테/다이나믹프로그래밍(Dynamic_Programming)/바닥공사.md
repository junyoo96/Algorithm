# 문제 유형
- 다이나믹 프로그래밍

# 주요 코드 개념
- 점화식 세우기
  - 2가지 경우만 고려하면 됨    
    1. 왼쪽부터 i - 1 까지 길이가 덮개로 이미 채워져 있으면, 하나의 경우만 존재 
      - 1 x 2 덮개를 덯는 경우 
    2. 왼쪽부터 i - 2 까지 길이가 덮개로 이미 채워져 있으면, 2가지 경우가 존재
      - 2 x 1 덮개를 2개 넣는 경우 
      - 2 x 2 덮개를 하나를 넣은 경우 
  - 왼쪽부터 N - 2 미만의 길이에 대해서는 고려할 필요가 없음 
    - 사용할 수 있는 덮개의 형태가 최대 2 x 2의 직사각형 형태이기 때문에 
  - 위의 개념을 점화식으로 표현하면, 
    - ai = ai-1 + ai-2 * 2
      - ai = ai=1 + ai-2 + ai-2를 줄여서 표현한 것
