# 문제 유형 
- 다이나믹 프로그래밍 
  - 삼각형의 맨 위층부터 아래에 있는 수중 하나를 선택하며 아래 층으로 내려올 때 선택된 수의 합이 최대인 것을 찾는 문제
    - 각 칸마다 선택되어 수들의 최대 합을 저장하기 위해 다이나믹 프로그래밍 사용
  - Bottom-Up 방식으로 구현
    - 반복문과 DP 테이블 사용
